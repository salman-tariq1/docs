## How to Use

1. **Copy** any of the JSON examples below
2. **Go to the API playground** for the Marketplace Searches endpoint
3. **Paste** the JSON into the request body field
4. **Click "Try it now"** to execute the request

## Examples

### Count Marketplace Search Records
```json
{
  "module": "marketplace_searches__c",
  "count_only": "1"
}
```

### List All Marketplace Searches
```json
{
  "module": "marketplace_searches__c"
}
```

### List Marketplace Searches with Specific Fields
```json
{
  "module": "marketplace_searches__c",
  "filters": {
    "fields": ["name", "createddate", "search_type__c"],
    "order_by": "name:DESC",
    "max_num": "-1"
  }
}
```
