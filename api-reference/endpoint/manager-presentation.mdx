## How to Use

1. **Copy** any of the JSON examples below
2. **Go to the API playground** for the Manager Presentation endpoint
3. **Paste** the JSON into the request body field
4. **Click "Try it now"** to execute the request

## Examples

### Count Manager Presentation Records
```json
{
  "module": "manager_presentation__c",
  "count_only": "1"
}
```

### List All Manager Presentations
```json
{
  "module": "manager_presentation__c"
}
```

### List Manager Presentations with Specific Fields
```json
{
  "module": "manager_presentation__c",
  "filters": {
    "fields": ["name", "createddate", "presentation_type__c"],
    "order_by": "name:DESC",
    "max_num": "-1"
  }
}
```
