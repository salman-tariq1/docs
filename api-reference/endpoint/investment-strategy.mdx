## How to Use

1. **Copy** any of the JSON examples below
2. **Go to the API playground** for the Investment Strategy endpoint
3. **Paste** the JSON into the request body field
4. **Click "Try it now"** to execute the request

## Examples

### Count Investment Strategy Records
```json
{
  "module": "investment_strategy__c",
  "count_only": "1"
}
```

### List All Investment Strategies
```json
{
  "module": "investment_strategy__c"
}
```

### List Investment Strategies with Specific Fields
```json
{
  "module": "investment_strategy__c",
  "filters": {
    "fields": ["name", "createddate", "strategy_type__c"],
    "order_by": "name:DESC",
    "max_num": "-1"
  }
}
```
